# Rapports et rendus — Projet classification e-commerce

Ce dossier contient les rapports détaillés et le référentiel du projet de classification de produits Rakuten (texte, image, fusion multimodale). Ces documents sont conçus pour servir de base au **rapport final** et sont rédigés de façon accessible aux lecteurs non techniques.

## Fichiers

| Fichier | Description |
|---------|-------------|
| **reference_classes.md** | Référentiel des 27 classes produits (prdtypecode), superclasses, confusions fréquentes, légendes des colonnes |
| **rapport_partie_texte.md** | Rapport détaillé de la partie classification textuelle : méthodologie, résultats chiffrés, métriques, glossaire |
| **rapport_partie_image.md** | Rapport détaillé de la partie classification par images : ResNet50, CLIP, EfficientNet, résultats par classe |
| **rapport_partie_fusion.md** | Rapport détaillé de la fusion multimodale : late fusion, CLIP, matching, Recall@K |

## Pour les rédacteurs du rapport final

Chaque rapport contient :
- Une **synthèse** en fin de document pour faciliter la rédaction de la conclusion.
- Un **glossaire** des termes techniques (texte, image, fusion).
- Des **tableaux de résultats** avec les métriques observées.
- Des **recommandations** et **limites identifiées**.
